# Cadillac Formula 1® Team - Official Concept Website

![Cadillac F1](public/hero-image.png)

Welcome to the official concept website for the **Cadillac Formula 1® Team**. This project is a highly-polished, cinematic Next.js web application designed to showcase the new F1 car, drivers, team branding, and race calendar. 

## 🏎️ Features

- **Cinematic Hero Animation**: A stunning, scroll-driven visual experience showcasing the Cadillac F1 car in exquisite detail.
- **Dynamic Content**: Beautifully interactive car showcase, driver profiles, and team branding elements.
- **Modern Tech Stack**: Built using Next.js, React, Tailwind CSS, and Framer Motion for smooth, high-performance animations.
- **Responsive Design**: Designed to look perfect on both desktop and mobile devices.

## 🚀 Getting Started Locally

First, install the dependencies if you haven't already:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🌐 Deploying the Website Live

This repository contains the source code. If you want to make the website live on the internet so anyone can visit it, the easiest way is to use **Vercel**:

1. Go to [Vercel.com](https://vercel.com) and sign up with your GitHub account.
2. Click **Add New** -> **Project**.
3. Import this `Cadillac.F1` repository.
4. Click **Deploy**. Vercel will build your website and give you a live URL!
